# Indexs of Fox.ONE OpenSDK

> indexs for all foxone opensdk 


Fox.ONE OpenSDK 项目是为了区块链钱包、资产及交易提供的一体化解决方案。

## 现实需求

基于区块链技术的新兴商户、平台及用户对支付交易解决方案有着迫切且相同的需求


* 钱包账户
* 资产管理
* 快速交易
* 历史记录追溯
* 快速通知触达
* 系统安全与使用简易
